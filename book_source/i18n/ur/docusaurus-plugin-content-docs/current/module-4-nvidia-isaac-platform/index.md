# ماڈیول 4: NVIDIA ایسیک پلیٹ فارم

آخری ماڈیول NVIDIA ایسیک پلیٹ فارم کو متعارف کراتا ہے، جو روبوٹکس کی ترقی کے لیے اوزاروں کا ایک طاقتور مجموعہ ہے۔ ہم سیمولیشن کے لیے ایسیک سیم، تیز رفتار روبوٹکس ایپلی کیشنز کے لیے ایسیک ROS، وژوئل SLAM (VSLAM) اور نیویگیشن کی تکنیکوں کو احاطہ کریں گے، جو کیپسٹون پروجیکٹ میں اختتام پذیر ہوگا۔
